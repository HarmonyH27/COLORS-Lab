# COLORS-Lab
COLORS Lab project  for COP 4331

## Description
A website for list of colors. This web site lets users sign in, once signed in they can add and search for colors. 

## Techonologies
- Digital Ocean
- GoDaddy
- PuTTY
- FileZilla
- Visual Studios Code

## Instructions
-  Create a Lamp droplet using Digital Ocean
-  Obtain domain name for website using GoDaddy
-  Connect Digital Ocean domain(IP address) to new obtained domain from GoDaddy
-  Set up mySQL database
-  Create API endpoints with PHP
-  Create frontend with HTML and Javascript
-  Upload fronend and backend files to server
